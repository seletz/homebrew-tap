# Homebrew Tap

Custom Homebrew tap for [seletz](https://github.com/seletz) projects.

## Installation

```bash
brew tap seletz/tap
brew install odoo-work-cli
```

Or in one line:

```bash
brew install seletz/tap/odoo-work-cli
```

## Available Casks

| Cask            | Description                              |
| --------------- | ---------------------------------------- |
| `odoo-work-cli` | CLI tool for managing Odoo 17 timesheets |
